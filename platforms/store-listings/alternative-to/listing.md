## Full Description
EnigmAgent is an open-source, local-first secret vault designed for AI agent developers.

The core idea: instead of hardcoding API keys, tokens, and passwords in AI prompts and scripts — store them in an AES-256-GCM encrypted vault and reference them as {{PLACEHOLDER}} symbols. The vault resolves them at execution time. Nothing leaves your machine.

**Key features:**
- Local AES-256-GCM vault with Argon2id key derivation
- {{PLACEHOLDER}} reference system — symbolic in prompts, real at execution
- Integrates with 40+ AI frameworks: LangChain, CrewAI, AutoGen, LlamaIndex, n8n, Semantic Kernel, Haystack, SmolAgents, Phidata, and more
- VS Code, JetBrains, and browser extensions
- MCP server compatible (works with Claude, Cursor, Windsurf)
- Python SDK + TypeScript SDK
- Zero telemetry, fully offline

**Alternatives it competes with:** HashiCorp Vault (for local dev), 1Password CLI, Doppler, direnv, dotenv
