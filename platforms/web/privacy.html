Privacy Policy

Last updated: April 2026

EnigmAgent ("the Extension") is an open-source, local-first credential vault for AI developers. This policy explains how the Extension handles user data.

1. Data We Collect

The Extension stores the following data exclusively on the user's local device:

  • Credential names, domain bindings, and encrypted credential values entered by the user into the vault.
  • Vault unlock state and extension preferences via chrome.storage.local.

No data is collected, transmitted, or stored on any external server. No analytics, telemetry, or tracking of any kind is performed.

2. How Data Is Used

  • Credential values are encrypted with AES-256-GCM using an Argon2id-derived key before being written to disk.
  • The tabs permission is used solely to read the current tab's URL for domain-binding verification — no tab content is read or stored.
  • The host permission is used only to run a content script that resolves {{PLACEHOLDER}} references on the active page at the user's explicit request.

3. Data Sharing

We do not sell, share, or transfer any user data to third parties. All data remains on the user's local machine at all times.

4. Data Retention

Data is retained only as long as the user keeps the encrypted vault file on their device. Users can delete all data at any time by removing the vault file.

5. Remote Code

The Extension does not load or execute any remote code. All JavaScript is bundled within the extension package.

6. Open Source

The full source code is available at github.com/agnuxo1/EnigmAgent for independent security auditing.

7. Contact

For questions or concerns, open an issue at github.com/agnuxo1/EnigmAgent/issues.
